## Setup

To set up and run this project locally, follow these steps:

1. **Clone the repository:**

    ```bash
    git clone https://github.com/PreethamNaik12/portfolio.git
    cd portfolio
    ```

2. **Install dependencies:**

    ```bash
    npm install
    ```

3. **Run the development server:**

    ```bash
    npm start
    ```
